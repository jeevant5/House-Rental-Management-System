# 🏠 House Rental Management System

A web-based **House Rental Management System** built using **PHP, MySQL, HTML, CSS, and XAMPP**.  
This project helps manage rental properties, tenants, and bookings efficiently.  

---

## 📌 Features
- 🏘️ Add, update, and delete rental properties  
- 👤 Manage tenants and their details  
- 📅 Track bookings and rental history  
- 🔑 User authentication (Admin & Tenant login)  
- 📊 Dashboard for quick overview  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP (Apache & MySQL)  

---

## 🚀 Installation Guide

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/YOUR-USERNAME/house-rental-management-system.git
````

### 2️⃣ Move to XAMPP `htdocs` Folder

Place the project inside:

```
C:\xampp\htdocs\house_rental
```

### 3️⃣ Import Database

1. Open **phpMyAdmin** (`http://localhost/phpmyadmin/`)
2. Create a new database, e.g., `house_rental`
3. Import the SQL file located in `/database/house_rental.sql`

### 4️⃣ Run the Project

Start **Apache** and **MySQL** in XAMPP, then open:

```
http://localhost/house_rental/
```
